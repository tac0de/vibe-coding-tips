# Tailwind Cleanup

## 상황

Tailwind 클래스가 지저분하고 UI 위계가 흐려져 있다.

## 바로 복붙할 프롬프트

```txt
현재 Tailwind UI를 정리해라.

목표:
- spacing hierarchy 정리
- typography hierarchy 정리
- 반복 클래스 줄이기
- surface / border / shadow 사용 규칙 정리

금지:
- utility를 무조건 추상화하기
- 새 컴포넌트 남발
- 클래스만 길게 추가하기

수정 후 어떤 위계가 명확해졌는지만 짧게 설명해라.
```

## 좋은 출력의 기준

- 위계가 더 읽힌다

## 실패 패턴

- class string만 더 길어진다
