# Create Design Brief

## 상황

에이전트가 자꾸 양산형 UI를 만든다.

## 바로 복붙할 프롬프트

```txt
먼저 디자인 브리프를 만들고 시작해라.

필수:
- reference 3개
- 금지할 UI 문법 5개
- 색상 규칙 1개
- 레이아웃 원칙 3개
- mobile-first 원칙

구현은 하지 말고 브리프만 써라.
```

## 좋은 출력의 기준

- 금지와 허용이 분명하다

## 실패 패턴

- reference만 있고 금지사항이 없다
