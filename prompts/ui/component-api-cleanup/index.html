기대 출력

  • props와 variant 구조가 더 읽기 쉬워진다
  • 중복 클래스와 boolean 조합이 줄어든다

주의 포인트

  • 컴포넌트를 통째로 갈아엎는다
  • 새로운 추상 레이어를 과하게 만든다
