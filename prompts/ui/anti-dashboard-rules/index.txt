13:T4c9,<h2 id="situation">situation</h2>
<p>The landing or editorial UI keeps appearing like a SaaS dashboard.</p>
<h2 id="prompt-to-copy-and-paste-immediately">Prompt to copy and paste immediately</h2>
<pre><code class="language-txt">이번 UI 작업은 아래 anti-dashboard rules를 강하게 적용해라.

금지:
- KPI cards
- repeated pill groups
- generic SaaS dashboard tone
- glassmorphism
- equal-size feature blocks
- over-rounded playful surfaces

지켜야 할 것:
- strong visual hierarchy
- fewer surfaces
- one dominant stage
- tighter type rhythm
- mobile-first

수정 후 답변 형식:
1. 어떤 대시보드 문법을 제거했는지
2. 장면의 중심이 무엇인지
3. 아직 남은 위험한 문법
</code></pre>
<h2 id="criteria-for-good-output">Criteria for good output</h2>
<ul>
<li>The scene is read first.</li>
<li>Cards and pills are not the main characters</li>
<li>UI moves away from product dashboard</li>
</ul>
<h2 id="failure-pattern">failure pattern</h2>
<ul>
<li>Just change the color and leave the structure the same.</li>
<li>Do not reduce the number of cards</li>
<li>There is no dominant stage</li>
</ul>
