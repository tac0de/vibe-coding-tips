# Landing First Screen

## 상황

랜딩 첫 화면만 빠르게 잡고 싶다.

## 바로 복붙할 프롬프트

```txt
이번 패치는 랜딩 첫 화면만 만든다.

목표:
- 헤드라인 1개
- supporting line 1개
- dominant visual stage 1개
- mobile-first

금지:
- KPI cards
- SaaS dashboard tone
- repeated pill groups
- glassmorphism

참조:
- Reuters Graphics
- NYT interactive

수정 후:
1. 바꾼 파일
2. 장면이 어떻게 읽히는지
3. 남은 리스크
```

## 좋은 출력의 기준

- 장면이 주인공이다

## 실패 패턴

- 버튼과 카드가 주인공이다

## 다음 액션

browser verifier로 첫 인상을 본다.
