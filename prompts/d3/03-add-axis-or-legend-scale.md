# 03. Add Axis Or Legend Scale

## 상황

사용자가 값을 해석할 기준선을 읽을 수 있게 해야 한다.

## 바로 복붙할 프롬프트

```txt
이번 단계에서는 axis 또는 legend-scale만 설계해라.

목표:
- 사용자가 현재 값을 읽을 수 있게
- 메인 장면을 가리지 않게

기본 차트면 axis를,
장면형 시각화면 inset legend-scale을 우선 검토해라.
```

## 좋은 출력의 기준

- 해석 기준이 생긴다

## 실패 패턴

- 장면형 시각화에 큰 axis를 넣는다
