# 02. Design Scales

## 상황

데이터를 시각 강도, 위치, 크기로 바꾸는 기준을 설계하고 싶다.

## 바로 복붙할 프롬프트

```txt
이번 단계에서는 scale만 설계해라.

입력 데이터:
- [숫자 필드]

해야 할 일:
- 어떤 scale이 맞는지 선택
- domain / range 제안
- 왜 linear / band / time / sqrt 중 무엇을 쓰는지 설명

지금은 렌더링 금지
```

## 좋은 출력의 기준

- scale 선택 이유가 분명하다

## 실패 패턴

- 무조건 linear만 쓴다
