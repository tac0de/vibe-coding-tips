# 11. Annotation And Storytelling

## 상황

D3가 정보는 보여주지만 기사형 설명이나 스토리텔링 포인트가 약하다.

## 바로 복붙할 프롬프트

```txt
이번 단계에서는 차트 자체를 더 복잡하게 만들지 말고, annotation과 storytelling layer를 추가해라.

목표:
- callout 1~3개
- source label
- 짧은 explanatory caption

답변 형식:
1. 무엇을 annotation할지
2. 왜 그 지점이 중요한지
3. caption 문장 초안
4. source label 위치와 방식

주의:
- 긴 본문 패널 금지
- 데이터보다 주석이 더 시끄러워지면 안 된다
```

## 좋은 출력의 기준

- annotation이 시선을 안내한다
- caption이 짧고 기사형이다
- 출처가 자연스럽게 붙는다

## 실패 패턴

- callout을 너무 많이 넣는다
- 긴 설명 패널을 연다
- source label 없이 감으로 서사를 붙인다
