# Prompt File Template

## 상황

이 프롬프트를 언제 쓰는지 한두 문장으로 쓴다.

## 바로 복붙할 프롬프트

```txt
이번 단계의 목표를 한 문장으로 적는다.

목표:
- [목표]

제약:
- [제약]

금지:
- [금지사항]

답변 형식:
1. [항목]
2. [항목]
3. [항목]
4. [항목]
```

## 좋은 출력의 기준

- 기준 1
- 기준 2

## 실패 패턴

- 패턴 1
- 패턴 2
