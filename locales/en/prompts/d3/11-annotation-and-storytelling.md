# 11. Annotation And Storytelling

## situation

D3 shows information, but article-type explanations and storytelling points are weak.

## Prompt to copy and paste immediately

```txt
이번 단계에서는 차트 자체를 더 복잡하게 만들지 말고, annotation과 storytelling layer를 추가해라.

목표:
- callout 1~3개
- source label
- 짧은 explanatory caption

답변 형식:
1. 무엇을 annotation할지
2. 왜 그 지점이 중요한지
3. caption 문장 초안
4. source label 위치와 방식

주의:
- 긴 본문 패널 금지
- 데이터보다 주석이 더 시끄러워지면 안 된다
```

## Criteria for good output

- Annotation guides the gaze
- The caption is short and article-like.
- Source comes naturally

## failure pattern

- Inserting too many callouts
- Opens a long description panel
- Write a narrative by feel, without a source label
