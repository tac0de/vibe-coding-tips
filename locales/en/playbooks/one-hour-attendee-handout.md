## Rules to remember today

- If you don't lock the range, you'll hallucinate.
- Separate builder and reviewer
- Browser verification is done last.
- UI locks the scene first
- D3 is used only when interaction is needed.

## Common Mistakes

- Entrust structure reading and implementation to the builder at the same time
- Just make the UI “pretty”
- Making D3 big from the beginning
- Finished without a reviewer
